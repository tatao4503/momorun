🛡️ 모모런 개인정보처리방침

모모런 개발자(이하 "개발자")는 사용자의 개인정보를 매우 소중하게 생각하며, 본 애플리케이션(이하 "앱")의 이용과 관련하여 사용자의 권리를 보호하기 위해 최선을 다하고 있습니다.

1. 개인정보의 수집 및 이용 목적

본 앱은 기본적으로 사용자의 개인정보를 서버로 수집하거나 제3자에게 전송하지 않습니다. 모든 데이터는 사용자의 기기 내 로컬 저장소(localStorage)에만 안전하게 기록 및 보관됩니다.

2. 수집하는 정보의 종류 및 목적

  • 동작 센서 및 HealthKit 걸음 수 데이터: 실시간 걸음 수 측정, 주간 리포트 분석, 업적 달성 검증을 위해 사용되며 기기 밖으로 절대 전송되지 않습니다.
  • 위치 정보 (선택): 현재 날씨와 모모톡의 상황별 대사를 연동하기 위해 임시 사용되며, 서버에 저장되지 않고 API 호출 후 즉시 파기됩니다.
  • API Keys (Gemini, ElevenLabs): AI 대화 및 목소리 합성을 위해 사용자가 직접 설정 창에 입력한 키이며, 로컬 기기에만 보관되며 외부의 특정 중개 서버를 거치지 않고 직접 해당 API에 전송됩니다.

3. 개인정보의 보유 및 파기

앱을 삭제하거나 설정 내의 'Factory Reset (앱 전체 초기화)' 기능을 수행할 경우, 기기에 저장된 모든 신체 데이터, 걸음 기록, API Key 데이터가 영구히 파기됩니다.

4. 아동의 개인정보 보호

본 앱은 비상업적인 목적으로 제작된 순수 팬 게임이며, 13세 미만 아동의 건강 정보나 개인식별정보를 동의 없이 수집하지 않습니다.

5. 문의 사항

개인정보 처리 및 보안 관련 의문점이 있으신 경우, 제작자 이메일(taeung) 혹은 공식 깃허브 저장소 이슈를 통해 언제든 연락해 주시기 바랍니다.
