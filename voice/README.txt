모모런 음성 파일(.mp3) 가이드
=================================

이 폴더에 아래의 파일 이름으로 실제 노아의 보이스(.mp3) 파일을 넣어주시면 앱에서 자동으로 재생됩니다. 
파일이 없으면 브라우저의 기본 기계음(TTS)으로 대체되어 나옵니다.

[필요한 파일명과 상황]
- greeting1.mp3 : "기록할 준비됐어요. 오늘도 시작할까요, 선생님?"
- greeting2.mp3 : "선생님의 걸음, 한 걸음도 빠짐없이 기억해 둘게요."
- greeting3.mp3 : "오늘은 몇 보까지 걸으실 생각이세요?"
- start1.mp3    : "측정 시작할게요. 기록은 제게 맡기세요."
- start2.mp3    : "한 걸음도 놓치지 않고 적어 둘게요."
- start3.mp3    : "기록대로, 완벽하게. 가볼까요, 선생님?"
- stop1.mp3     : "여기까지 기록해 뒀어요. 언제든 이어서 하시면 돼요."
- stop2.mp3     : "잠깐 쉬어가도 괜찮아요. 기록은 그대로 남아 있어요."
- mile1.mp3     : "좋은 출발이에요, 선생님. 순조롭게 적히고 있어요."
- mile2.mp3     : "절반 지점이에요. 기록은 정확하게 남기고 있어요."
- mile3.mp3     : "조금만 더예요. 끝까지 함께 기록할게요."
- goal.mp3      : "수고하셨습니다, 선생님." (목표 달성 시)
- voice_on.mp3  : "음성을 켰어요, 선생님."

주의사항: 파일 확장자는 소문자 .mp3로 맞춰주세요.
