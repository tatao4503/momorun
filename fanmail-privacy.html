팬레터 데모 개인정보 안내

이 페이지는 공식 제출 전 오해를 줄이기 위해, 메일 공유용 데모의 데이터 사용 범위만 따로 정리한 안내입니다.

서버 전송 없음 팬레터 데모는 계정 로그인, 자체 서버, 원격 데이터베이스를 사용하지 않습니다.
로컬 저장 걸음 수와 최근 7일 기록은 현재 브라우저의 localStorage에만 저장됩니다.
센서 권한 기록 시작 버튼을 누른 뒤 브라우저가 허용하는 경우에만 동작 센서를 사용합니다.
샘플 기록 샘플 기록 추가 버튼은 화면 설명용 가상 걸음 수를 현재 브라우저에만 더합니다.
제외 기능 메일용 데모에는 결제, 광고, 게임 재화, AI API Key, 음성 합성, 위치 기반 날씨 기능을 포함하지 않았습니다.

Blue Archive 및 관련 캐릭터 권리는 각 권리자에게 있으며, 본 데모는 비공식 비상업 팬 창작물입니다.
